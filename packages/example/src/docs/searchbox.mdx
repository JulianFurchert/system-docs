```js live removeFragment
  return (
    <div>
      <Searchbox onSubmit={(value) => setSearchTerm(value)}/>
      <Flex mt={2}>
        <Text mr={2}>
          Last Searchterm:
        </Text>
        <Text fontWeight={600}>
          {searchTerm}
        </Text>
      </Flex>
    </div>
  );
};
```

## View customizations
You can customize the entire view. The returned functions getInputProps and getSubmitProps are prop getters. They are return a props object to spread over their corresponding UI elements. This lets you arrange elements however you'd like in the DOM.

### With icons

```js live
<Searchbox autoComplete sayt>
  {({suggestions, results}) => (
    <React.Fragment>
      <Flex>
        <SearchboxInput autocomplete sayt />
        <SearchboxButton>
          Search
        </SearchboxButton>
      </Flex>
      <SearchboxPopover portal={false}>
        <SearchboxList aria-labelledby="demo">
          {suggestions.autocomplete && 
          suggestions.autocomplete.slice(0, 3).map(value => (
              <SearchboxSuggestion value={value}>
                <Icon
                  size={14} 
                  color="textSecondary" 
                  className="fal fa-search" 
                  mr={2}
                /> 
                <SearchboxTextHighlight />
              </SearchboxSuggestion>
            ))}
            {results.results && 
            results.results.slice(0, 3).map(result => (
              <SearchboxLink title={result.title[0]} href={result.url} >
                <Icon
                  size={14} 
                  color="textSecondary" 
                  className="fal fa-link" 
                  mr={2}
                /> 
                <SearchboxTextHighlight />
              </SearchboxLink>
            ))}
        </SearchboxList>
      </SearchboxPopover>
    </React.Fragment>
  )}
</Searchbox>
```
