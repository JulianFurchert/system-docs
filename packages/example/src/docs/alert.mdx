### Sizes

```js live
<button>Sizes</button>
```

### Waiting state

If true, when the user clicks inside the text box the current value will be selected. Use this if the user is likely to delete all the text anyway (like the URL bar in browsers)

```js live
<button>Waiting</button>
```
